Extended Kalman Filter

E05 Optimal Filtering Nonlinear Kalman 2-Link Arm Est. 1960

Real robots are nonlinear. The EKF handles this through local linearization via Jacobians evaluated at the current state estimate.

Prerequisite: E04: Kalman Filter

Overview

The standard Kalman filter assumes linear dynamics $\mathbf{x}_{k+1}=\mathbf{A}\mathbf{x}_k$. Real robot dynamics are nonlinear: $\mathbf{x}_{k+1}=f(\mathbf{x}_k)$. The EKF linearizes $f$ around the current estimate using the Jacobian $\mathbf{F}_k = \partial f/\partial \mathbf{x}\big|_{\hat{\mathbf{x}}_k}$, then runs a standard Kalman filter with this time-varying linearization.

Key Insight

The EKF is only a first-order approximation. It can diverge when nonlinearity is strong and the estimate drifts far from truth, because the linearization becomes invalid. This motivates the particle filter (E09) for highly nonlinear problems.

Key Properties

  • Local linearization — Jacobian computed at each step
  • Not globally optimal — Only locally optimal near the true state
  • Can diverge — If nonlinearity is too strong for the linearization
  • Computationally efficient — Same cost as linear KF per step

How It Works

  1. Predict state: $\hat{\mathbf{x}}^{-} = f(\hat{\mathbf{x}}^{+})$ — Propagate through the nonlinear model
  2. Compute Jacobian: $\mathbf{F}_k = \frac{\partial f}{\partial \mathbf{x}}\bigg|_{\hat{\mathbf{x}}^{+}}$ — Linearize at current estimate
  3. Predict covariance: $\mathbf{P}^{-} = \mathbf{F}_k\mathbf{P}^{+}\mathbf{F}_k^T + \mathbf{Q}$
  4. Update: Same as standard Kalman filter using the linearized system

Key Equations

Nonlinear State Prediction

$$\hat{\mathbf{x}}_{k+1}^{-} = f(\hat{\mathbf{x}}_k^{+}, \mathbf{u}_k)$$

Jacobian (State Transition)

$$\mathbf{F}_k = \frac{\partial f}{\partial \mathbf{x}}\bigg|_{\hat{\mathbf{x}}_k^{+}} \approx \begin{bmatrix}\frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n}\\ \vdots & \ddots & \vdots \\ \frac{\partial f_n}{\partial x_1} & \cdots & \frac{\partial f_n}{\partial x_n}\end{bmatrix}$$

Linearized Covariance Propagation

$$\mathbf{P}_{k+1}^{-} = \mathbf{F}_k \mathbf{P}_k^{+} \mathbf{F}_k^T + \mathbf{Q}$$

MATLAB Implementation

MATLAB
%% E05: Extended Kalman Filter (2-Link Arm)
function xnext = armDynamics(x, dt)
    q1=x(1); q2=x(2); qd1=x(3); qd2=x(4);
    g=9.81; L1=1; L2=0.8; m1=1; m2=0.8; d=0.1;
    qdd1 = (-(m1+m2)*g*L1*sin(q1) - ...
            m2*g*L2*sin(q1+q2)*0.3 - d*qd1) / ((m1+m2)*L1^2);
    qdd2 = (-m2*g*L2*sin(q1+q2) - d*qd2) / (m2*L2^2);
    xnext = x + dt*[qd1; qd2; qdd1; qdd2];
end

% Numerical Jacobian
function F = numJacobian(f, x, dt)
    n = length(x); eps = 1e-5; f0 = f(x,dt);
    F = zeros(n);
    for j = 1:n
        xp = x; xp(j) = xp(j) + eps;
        F(:,j) = (f(xp,dt) - f0) / eps;
    end
end

% EKF loop
x = [0.8;-0.5;0.5;-0.3]; xhat = zeros(4,1);
P = eye(4); Q = 0.005*eye(4); R = 0.05*eye(2);
C = [eye(2) zeros(2)];
for k = 1:250
    y = C*x + chol(R)'*randn(2,1);
    F = numJacobian(@armDynamics, xhat, dt);
    xhat_m = armDynamics(xhat, dt);
    P_m = F*P*F' + Q;
    S = C*P_m*C' + R;
    K = P_m*C'/S;
    xhat = xhat_m + K*(y - C*xhat_m);
    P = (eye(4)-K*C)*P_m;
    x = armDynamics(x, dt) + chol(Q)'*randn(4,1);
end
